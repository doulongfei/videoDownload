使用演示

简单几步，快速上手

1

启动服务器

终端 
cd backend
go run ./cmd/main.go

# 输出：
# 2024/01/15 10:30:00 Server starting on :5000
# 2024/01/15 10:30:00 Static files served from: /frontend
2

分析视频资源

API调用 
curl -X POST http://localhost:5000/api/analyze \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com/video-page"}'
3

创建下载任务

下载请求 
curl -X POST http://localhost:5000/api/download \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://example.com/video.m3u8",
    "filename": "my-video"
  }'

API文档

完整的RESTful API接口

核心端点

方法 端点 描述
POST /api/analyze 分析网页提取视频
POST /api/download 创建下载任务
GET /api/status 获取所有任务状态
GET /api/status/{id} 获取指定任务状态
SSE /api/progress/{id} 实时进度流

响应示例

分析响应 
{
  "success": true,
  "videos": [
    {
      "url": "https://example.com/video.m3u8",
      "type": "m3u8",
      "quality": "1080p",
      "title": "示例视频",
      "duration": "00:10:30"
    }
  ]
}

}

安装指南

快速开始使用

环境要求

🔧
Go 1.24.4+
Go语言运行环境
🎬
FFmpeg
视频处理工具

安装步骤

1

克隆项目

git clone https://github.com/doulongfei/videoDownload.git
cd videoDownload
2

安装依赖

cd backend
go mod download
3

启动服务

# 编译运行
go build -o server ./cmd && ./server

# 或直接运行
go run ./cmd/main.go
4

访问界面

打开浏览器访问: http://localhost:5000
